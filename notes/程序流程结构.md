3、接着是do...while循环结构

4、然后用水仙花数这个实例来展示这个结构的运用
```cpp
#include<iostream>
using namespace std;

{
	//先打印所有的水仙花数
	int i = 100;
	do
	{
	//获取该数字的个位，十位，百位
	int ge = i % 10;

	int bai = i / 100;

```

7、再然后是嵌套循环
```cpp
#include<iostream>
using namespace std;

int main() {
	
	for (int i = 0 ; i < 10; i++)
	{
		cout << "* ";
	}
	return 0;
}
```
首先你看到这个的输出结果只有一行*号（因为在这里没有使用换行符，所以都在一行中呈现），再然后我想达到一个10*10矩阵的效果，所以我最好能够将这个循环再次执行10次，因为我不可能去把这行代码去复制10次，所以就有了嵌套循环这种方式
```cpp
#include<iostream>
using namespace std;

int main() {
	
	for (int i = 0 ; i < 10; i++)
	{
		for (int i = 0; i < 10; i++)
		{
			cout << "* ";
		}
		cout << endl;
	}
	return 0;
}
```
注意1：再有就是这个程序的最后一行必须要有cout<<endl这个代码，这表示这个循环结束一次后要进行一次换行，否则就不是10*10的类型了
注意2：还有就是最好嵌套中不要都用i，最好能用不同的字母，不然可能会造成一些误解
8、接下来用乘法口诀表的这个实例应用来呈现
```cpp
#include<iostream>
using namespace std;

int main() {
	
	for (int i = 1; i <= 9; i++)
	{
		for (int j = 1; j <= i; j++)
		{
			cout << i << "*" << j << "=" << i * j << "  ";
		}
		cout << endl;
	}
	return 0;
}
```
我觉得这个代码的核心难点第一个在于怎么实现那种类似于乘法口诀表那种斜着的情形，所以这个的实现主要就在于i<j这个代码，它的列数是小于等于行数的（以先输出列数的角度来看），第二个在于你对这个for循环以及嵌套循环的流程以及究竟输出的是什么结果得要有深刻的理解，不然的你都想不出来为啥是要这样写的

2、continue语句
作用：在循环中，跳过本次循环中尚未执行的语句，继续执行下一个循环
```cpp
#include<iostream>
using namespace std;

int main() {
	
	for (int i = 0; i <= 100; i++) {
		if (i % 2 == 0)
		{
			continue;
		}
		cout << i << endl;
	}
	return 0;
}
```
与break的区别是前者退出这次循环后会继续执行下一个循环，而后者就是直接退出整个循环了，以这段代码为例，如果用的是break就会输出不了任何值，而用continue的话就会只输出奇数
3、goto语句
作用：可以无条件的跳转，跑我们想要的代码
具体语法以下面这段代码为例：
```cpp
#include<iostream>
using namespace std;

int main() {
	
	cout<<"1.hello world"<<endl;
	goto FLAG;
	cout << "2.hello world"<< endl;
	cout << "3.hello world" << endl;
	cout << "4.hello world" << endl;
    FLAG:
	cout << "5.hello world" << endl;
	return 0;
}
```
一般不推荐使用，以免造成程序的流程混乱
