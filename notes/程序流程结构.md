一、顺序结构
1、首先是顺序循环结构，这里将单分支，多分支if语句，以及多条件if语句用三只小猪的例子来呈现
```cpp
#include<iostream>
using namespace std;
int main1()
{
	int a = 0;
	int b = 0;
	int c = 0;
	cout << "请输入小猪A的体重" << endl;
	cin >> a;
	cout << "请输入小猪B的体重" << endl;
	cin >> b;
	cout << "请输入小猪C的体重" << endl;
	cin >> c;
	
	if (a > b)
	{
		if (a > c)
		{
			cout << "A是体重最大的小猪" << endl;
		}
		else
		{
			cout << "C是体重最大的小猪" << endl;
		}

	}
	else
	{
		if (b > c)
		{
			cout << "B是体重最大的小猪" << endl;
		}
		else
		{
			cout << "C是体重最大的小猪" << endl;
		}
	}
	system("pause");

	return 0;
}
```

3、最后是switch语句
```cpp
#include<iostream>//switch语句的缺点不能输入范围值，只能输入整型或字符型，但结构清晰，执行的效率高

int main3(){
	cout<<"请给这个电影打分"<<endl;

	case 10:
		cout << "你认为是这部电影是经典电影" << endl;

        case 9:

		break;
        case 8:
		cout << "你认为是这部电影是中规中矩" << endl;
		break;
        case 7:
		cout << "你认为是这部电影是普通电影" << endl;
		break;
        case 6:
		cout << "你认为是这部电影是普通电影" << endl;
		break;
        case 5:
		cout << "你认为这部电影是不好看" << endl;

二、循环结构
1、首先用这个程序来展示while循环结构
```cpp
#include<iostream>
using namespace std;

int main4()
{
    int i = 0;
    while(i <= 9)//while的（）中是循环条件，满足条件则循环，否则结束循环，要避免陷入死循环
    {
        cout << i << endl;
        i++;
    }
    return 0;
}
```
2、然后用猜数字这个实例来展示

#include<iostream>

//还要包含一个含有time的头文件

3、接着是do...while循环结构
```cpp
#include<iostream>
using namespace std;

int main6()
{
	//do...while语句,该语句是先执行一次，再判断条件，如果条件为真，则继续执行，否则结束循环
	int i = 0;
	do
	{
		cout << "i=" << i << endl;
		i++;
	} while (i < 10);

    return 0;
```
4、然后用水仙花数这个实例来展示这个结构的运用
#include<iostream>
using namespace std;

int main()
{
	//先打印所有的水仙花数
	int i = 100;
	do
	{
	//获取该数字的个位，十位，百位
	int ge = i % 10;
	int shi = i / 10 % 10;
	int bai = i / 100;

	//判断该数字是否是水仙花数
	if (ge * ge * ge + shi * shi * shi + bai * bai * bai == i)
	{
		cout << i << endl;
	}
	i++; 
	} while (i < 1000);
    

	return 0;
}
