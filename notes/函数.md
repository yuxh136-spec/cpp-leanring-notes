#**函数**  
一、函数的作用  
代码复用  
避免重复编写相同功能的代码  
提高开发效率和代码可维护性  
模块化设计  
将复杂程序分解为小的、独立的模块  
便于团队协作和分模块调试

  函数体语句  
  return语句  
}  
三、函数的调用  
```cpp
#include<iostream>
using namespace std;
int add(int sum1, int sum2)
{
    int sum = sum1 + sum2;
    return sum;
};//(这里的a,b为形参）
int main()
{
    int a;
    int b;
    cout << "请输入两个数：" << endl;
    cin >> a >> b;//(这里的a,b为实参)
    cout << "两个数的和为：" << add(a, b) << endl;
    return 0;
}
```
四、值传递  
这里用值交换的实例来演示  
注意：如果一个函数体没有返回值的时候可以有一个void的声明  
```cpp
#include<iostream>
using namespace std;
void swap(int num1, int num2)
{
    int temp;
    temp = num1;
    num1 = num2;
    num2 = temp;
    cout << "交换后的数字为" << num1 << " " << num2 << endl;
}
int main()
{
    int a = 10, b = 20;
    swap(a, b);
    cout << "a=" << a << endl;
    cout << "b=" << b << endl;//当你在进行值传递的时候，形参会发生改变，但是实参不会改变
    return 0;
}
```
五、函数的常见样式  
```cpp
#include<iostream>
using namespace std;
//无参无返
void test01(){
    cout<<"无参无返"<<endl;
}
//有参无返
void test02(int a){
    cout<<"有参无返"<<a<<endl;
}
//无参有返
int test03(){
    cout<<"无参有返"<<endl;
    return 100;
}
//有参有返
int test04(int a){
    cout<<"有参有返"<<a<<endl;
    return a;
}
int main(){
    //无参无返的调用
    test01();
    //有参无返的调用
    test02(100);
    //无参有返的调用
    int ret = test03();
    cout<<"ret = "<<ret<<endl;
    //有参有返的调用
    int ret2 = test04(100);
    cout<<"ret2 = "<<ret2<<endl;
    return 0;
}
```
六、函数声明  
```cpp
#include<iostream>
using namespace std;
int max(int a, int b);

int main()
{ 
    int i = 10, j = 20;
    cout << max(i,j);
}

int max(int a, int b)
{
    return a > b ? a : b;
}
```
前几写代码的时候，都是函数在前，main函数在后面，如果单纯的换二者的顺序，就会出现报错，这时候就可以在前面写一个声明，
就如上述提供的代码所示，不需要把所有的东西都写出来，就是告诉编译器这个函数是存在的  
七、函数的分文件编写  
a.创建.h为后缀的头文件
b.创建.cpp为后缀的源文件
c.在头文件中写函数的声明  (max.h)
```cpp
#include<iostream>
using namespace std;

int max(int a, int b);
```
d.在源文件中写函数的定义  (mac.cpp)
```cpp
#include"max.h"

int max(int a, int b)
{
    return a > b ? a : b;
}
```
f.最后是调用  
```cpp
#include<iostream>
using namespace std;
#include"max.h"//在这里只用包含我们自己定义的头文件就行了

int main()
{
    cout<<max(1,2)<<endl;
}
```
