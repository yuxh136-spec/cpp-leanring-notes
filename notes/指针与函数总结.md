# 📌 指针与函数（Pointers & Functions）

## 一、指针作为函数参数的作用

当函数参数使用**指针类型**时，函数内部可以通过指针**间接访问并修改实参的值**。

> 👉 本质：把“变量的地址”传进函数，而不是把“变量的值”传进去。

---

## 二、值传递（Pass by Value）

### 1. 形式

```cpp
void swap1(int a, int b)
```

### 2. 特点

* 形参 `a`、`b` 是实参的**拷贝**
* 函数内部对 `a`、`b` 的修改
  👉 只作用于副本
  👉 **不会影响 main 中的变量**

### 3. 代码示例

```cpp
void swap1(int a, int b)
{
    int temp = a;
    a = b;
    b = temp;
}
```

### 4. 调用方式

```cpp
swap1(a, b);
```

### 5. 结果

```text
a = 10
b = 20
```

### 6. 内存示意

```text
main中的a -----> 10
main中的b -----> 20

swap1中的a -----> 10 (副本)
swap1中的b -----> 20 (副本)
```

➡️ 修改的是副本，不影响原变量。

---

## 三、地址传递（指针传参 / Pass by Address）

### 1. 形式

```cpp
void swap2(int* p1, int* p2)
```

### 2. 特点

* 形参是指针变量
* 存放的是实参的**地址**
* 通过 `*p1`、`*p2` 间接操作原变量

### 3. 代码示例

```cpp
void swap2(int* p1, int* p2)
{
    int temp = *p1;
    *p1 = *p2;
    *p2 = temp;
}
```

### 4. 调用方式

```cpp
swap2(&a, &b);
```

### 5. 结果

```text
a = 20
b = 10
```

### 6. 内存示意

```text
p1 -----> a的地址 -----> 10
p2 -----> b的地址 -----> 20
```

➡️ 通过指针修改了**原变量本身**。

---

## 四、核心对比

| 对比项     | 值传递  | 指针传递   |
| ------- | ---- | ------ |
| 传入内容    | 值的副本 | 变量地址   |
| 是否可修改实参 | ❌ 否  | ✅ 是    |
| 参数类型    | 普通变量 | 指针变量   |
| 使用场景    | 只读参数 | 需要修改实参 |

---

## 五、关键语法总结

### 1. 取地址符

```cpp
&a
```

表示：获取变量 a 的地址

---

### 2. 解引用运算符

```cpp
*p
```

表示：访问指针 p 指向地址中的数据

---

### 3. 记忆口诀

> &：拿地址
> *：取值

---

## 六、一句话总结

✅ 普通参数 —— 传值
❌ 不能改变实参

✅ 指针参数 —— 传地址
✅ 可以改变实参
